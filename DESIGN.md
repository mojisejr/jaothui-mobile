# JAOTHUI Mobile Design Contract

Phase 3 uses a native translation of the existing JAOTHUI v2 design language.

## Visual Grammar

- Atmosphere: dark-first, premium, Thai buffalo heritage.
- Primary accent: antique gold.
- Trust accent: green for verified/success states.
- Canvas: near-black.
- Surfaces: raised dark cards with soft gold borders.
- Imagery: real buffalo photos from the Mobile BFF API.

## Constraints

- Native components, not copied Tailwind classes.
- Touch targets should be at least 44px.
- Routes must stay usable before auth exists.
- Public journey only in Phase 3: Home, Buffalo list, Cert detail.

## Tokens

Canonical tokens live in `src/design/tokens.ts`.
